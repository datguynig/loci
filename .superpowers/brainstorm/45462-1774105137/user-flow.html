Loci — Core User Flow

From sign-up to reading in as few steps as possible. Agreed architecture: Clerk + Supabase.

1
Sign up / Sign in
Clerk modal appears on first visit. Email + password, or Google OAuth. On return visit, Clerk restores the session silently — no modal unless explicitly signed out. First-time users land on the empty library immediately after confirming email.
2
Upload first EPUB
Empty library shows a centred upload prompt — drag and drop or click to browse. File uploads to Supabase Storage. Cover extracted from the EPUB automatically. On success, the book opens directly in the reader. No intermediate confirmation screen.
3
Read or listen
Reader opens at the first chapter. User can read at their own pace, or hit Play to let TTS read aloud with sentence highlighting and autoscroll. All existing controls (font size, layout, theme, voice, speed) are immediately available.
↓
While reading — available actions
User can do any of these at any time:
Actions available in reader
Progress autosaves
Chapter, scroll position, and TTS sentence index saved to Supabase every 30s and on chapter change. No user action needed.
Bookmark
One tap to drop a pin at the current position. Appears in the sidebar. Tap to jump back to it on any device.
Add a note
Select any text → "+ Note" bubble → write note → saved to Supabase. Annotated passage gets warm underline in the text.
Search within book
Ctrl+F opens search bar. Jump to any match in the spine.
5
Close book → Library
Header back-arrow (or keyboard shortcut) returns to the library. Library shows all books with cover, progress %, and last-read date. Clicking any book resumes from the saved position automatically.
6
Notes — where do they live?
Notes already exist in the reader sidebar. The question is whether to surface them at the library level too.
Open question
Do you need a notes view outside the reader?
The reader sidebar already shows notes per-chapter. Two additional surfaces are possible — pick what's useful to you:

Per-book notes page: accessible from the library card. Shows all notes for that book as a clean reading list — quote in serif, your note beneath it, chapter label. Good for review before a meeting or exporting.

Global notes view: a "Notes" tab in the library. All annotations across your entire library, grouped by book, searchable. More like a Kindle highlights dashboard.

Three architectural directions

How the AI study system is structured — each has a different feel

A

Tabbed study panel

The right panel has explicit tabs: Chat · Summary · Flashcards · Notes. Each tab has its own dedicated UI — summary shows a generated digest, flashcards shows a card stack, notes shows the scratchpad. AI is a feature set, clearly organised.

Pros

  • Predictable, easy to learn
  • Each mode has purpose-built UI

Cons

  • Feels like a feature list, not a product
  • Context switching between tabs breaks flow
  • Most apps look like this
B

Chat-first with smart chips ✦ Recommended

The panel is a single, beautiful chat thread. Above the input, context-aware chips appear based on what you're doing: reading → "Summarise chapter". Notes open → "Review my notes". Flashcards are generated inline as cards within the chat thread and auto-saved. No tabs — the AI knows where you are.

Pros

  • Feels like a personal tutor, not a tool
  • One surface — no context switching
  • Chips make capabilities discoverable
  • Flashcards/summaries live in chat history
  • Visually distinctive — no app looks like this

Cons

  • Harder to build the chip logic
  • Flashcard review UI needs to be elegant within chat
C

Ambient AI (woven throughout)

No dedicated AI panel. Instead, AI is embedded everywhere: hover a highlight → "Explain", chapter header → "Summarise", selection bubble → "Add to flashcards", scratchpad → "Help me revise this". The AI is invisible infrastructure, not a destination.

Pros

  • Most "magical" feeling
  • Zero UI overhead
  • AI feels native, not bolted on

Cons

  • No place for open-ended Q&A
  • No persistent conversation
  • Hard to discover all capabilities
  • Very complex to implement well

Why B wins for an award-winning product: Tabs (A) look like every edtech tool built in 2019. Ambient-only (C) sacrifices the persistent conversation that makes AI feel like a tutor. B gives you a single, gorgeous chat surface that feels intelligent — it knows you're on Chapter 3, surfaces the right chip, generates a flashcard deck that drops into the thread as a beautiful card stack. The whole thing reads as one coherent experience, not a feature grid.
